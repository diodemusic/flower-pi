# flower-pi

## Installation

```bash
git clone https://github.com/diodemusic/flower-pi.git
cd flower-pi/
sudo chmod u+x init.sh
./init.sh
```

## Now spin up the API

```bash
sudo chmod u+x start.sh
./start.sh
```

You only need to run ./init.sh once
You also only need to run chmod on init.sh and start.sh once
